# RMK Group - Website Institucional

Este é o site institucional da **RMK Group**, desenvolvido com foco em excelência comercial, representação estratégica e treinamentos de vendas.

## 🚀 Tecnologias Utilizadas
- **HTML5** (Semântico)
- **CSS3** (Personalizado com variáveis de design system)
- **Bootstrap 5.3** (Grid e Componentes responsivos)
- **Vanilla JavaScript** (Navegação SPA e Animações de Scroll)

## 🎨 Identidade Visual
O design segue uma estética **Dark Metal** com detalhes em **Neon Blue, Deep Blue e Violet**, conforme a identidade visual da marca.

## 🛠️ Como Executar Localmente
1. Clone o repositório:
   ```bash
   git clone https://github.com/BrianWS01/RMKGROUP.git
   ```
2. Abra o arquivo `index.html` em seu navegador ou utilize um servidor local:
   ```bash
   npx http-server ./
   ```
